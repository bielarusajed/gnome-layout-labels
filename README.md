# Keyboard layouts labels

Simply change the layout labels for the keyboard layout indicator in the top panel to anything you want

Install here (when/if it get approved): https://extensions.gnome.org/extension/9156/keyboard-layouts-labels/

# Install from repo

```sh
gnome-extensions pack
gnome-extensions install layout-labels@bielarusajed.gay.shell-extension.zip
```
